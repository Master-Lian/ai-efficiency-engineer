1. utils/model_opt.py（模型构建 + 量化优化）
AI 效率工程师核心代码：包含模型定义、INT8 动态量化（超聚变必备优化手段）

2. test_perf.py（独立性能测试脚本）
验证模型优化效果，可直接运行，不依赖 Streamlit

3. utils/agent_core.py（Agent + MCP 调度中心）
整合模型创建、量化、性能测试三大技能，完全适配之前的调用逻辑

4. app.py（Streamlit 可视化主程序）
最终展示页面，整合所有功能，面试直接演示

5. utils/init.py
空文件即可（标识为 Python 包）


运行指令（最终验收）
    # 1. 独立测试性能（控制台查看优化效果）
    python test_perf.py

    # 2. 启动可视化Demo（面试展示用）
    streamlit run app.py

运行成功标准
1. 控制台脚本：输出原始模型 vs 量化模型性能对比
2. 网页端：
    一键创建模型
    一键量化优化
    一键性能测试
    可视化展示效率提升效果
