# 5G 运维知识库

## 5G NR 基础概念

### RSRP (Reference Signal Received Power)
参考信号接收功率，是衡量5G网络信号强度的关键指标。
- 优秀: > -80 dBm
- 良好: -80 ~ -90 dBm
- 一般: -90 ~ -100 dBm
- 差: < -100 dBm

### SINR (Signal to Interference plus Noise Ratio)
信号与干扰加噪声比，反映信号质量。
- 优秀: > 20 dB
- 良好: 13 ~ 20 dB
- 一般: 5 ~ 13 dB
- 差: < 5 dB

### 常见故障类型
1. **弱信号故障**: RSRP过低，导致连接不稳定
2. **干扰故障**: SINR过低，存在同频或邻频干扰
3. **切换失败**: 小区间切换参数配置不当
4. **吞吐量下降**: 资源调度或信道质量异常

## 故障处理建议

### 弱信号处理
- 调整天线倾角
- 增加发射功率
- 优化站点位置

### 干扰处理
- 调整PCI规划
- 优化频率复用
- 启用ICIC功能
